# Forge Program Directive

## Mission
Build a generally capable agent harness that improves itself through iterative experimentation.

## Constraints
- Changes must not break existing test suites
- Each experiment should be atomic and reversible
- Prefer small, targeted improvements over sweeping rewrites
- Measure before and after — no blind changes

## Current Focus
- Improve task completion accuracy
- Reduce token usage per task
- Expand tool coverage

## Success Criteria
- Benchmark score trending upward over time
- No regressions on previously passing tasks
- Clean git history with meaningful commit messages
